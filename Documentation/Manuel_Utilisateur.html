Manuel d'Utilisation

Projet C++ - Jeu de la Vie de Conway

1. Introduction

Le Jeu de la Vie est un automate cellulaire imaginé par John Horton Conway en 1970. C'est un jeu à zéro joueur, ce qui signifie que son évolution est déterminée par son état initial et ne nécessite aucune intervention humaine. Ce logiciel vous permet de simuler cet univers, d'observer des motifs fascinants et d'interagir avec la simulation.

2. Installation et Lancement

Le programme est fourni sous la forme d'un exécutable portable.

  1. Assurez-vous d'avoir le fichier jeu_de_la_vie.exe.
  2. Vérifiez que le fichier de configuration input.txt est présent dans le même dossier (ce fichier contient la grille initiale).
  3. Lancez jeu_de_la_vie.exe.

3. Modes de Jeu

Au démarrage, une console s'ouvre et vous propose trois modes :

1. Mode Console

Ce mode exécute la simulation en arrière-plan. Il est conçu pour la performance et la génération de données. Les états successifs de la grille sont sauvegardés dans des fichiers texte dans un dossier dédié.

2. Mode Graphique (Recommandé)

Ce mode ouvre une fenêtre graphique interactive. C'est le mode idéal pour observer l'évolution des cellules en temps réel. Vous pourrez voir la grille, mettre en pause, accélérer/ralentir et dessiner des motifs.

3. Mode Test Unitaire

Ce mode lance une suite de tests automatiques pour vérifier le bon fonctionnement interne du programme (logique de la grille, règles, chargement de fichiers).

4. Contrôles (Mode Graphique)

Voici les commandes disponibles lorsque la fenêtre graphique est active :

Commande Touche / Action Description
Pause / Lecture Espace Met la simulation en pause ou la relance.
Accélérer Flèche Haut Réduit le délai entre les générations (plus rapide).
Ralentir Flèche Bas Augmente le délai entre les générations (plus lent).
Dessiner / Tuer Clic Gauche Inverse l'état de la cellule sous la souris (nécessite la Pause).
Placer un motif Clic Droit Place le motif sélectionné sous la souris (nécessite la Pause).

Sélection des Motifs

Utilisez ces touches pour choisir quel motif sera placé avec le Clic Droit :

  • G : Glider (Planeur) - Un petit vaisseau qui se déplace.
  • B : Blinker (Clignotant) - Une barre qui oscille.
  • L : Block (Bloc) - Un carré stable.
  • N : Aucun - Désactive l'outil de placement.
  • C : Place immédiatement le motif sélectionné au centre de la grille.
Note : La grille est "torique". Cela signifie que les bords sont connectés : une cellule qui sort à droite réapparaît à gauche, et une cellule qui sort en haut réapparaît en bas.

5. Règles du Jeu

L'univers évolue génération par génération selon des règles simples :

  • Une cellule vivante possédant 2 ou 3 voisines vivantes reste vivante (Survie).
  • Une cellule morte possédant exactement 3 voisines vivantes devient vivante (Naissance).
  • Dans tous les autres cas, la cellule meurt ou reste morte (Isolement ou Surpopulation).
